The Programme Management System (PMS) is designed to help programme coordinators centrally manage academic operations. It provides a clean, intuitive interface for handling:

Student Management: Track personal details, year of study, and module registrations

Lecturer Management: Manage lecturer information and module assignments

Module Administration: Create and manage academic modules with credits

Registration System: Handle student module registrations

Assignment Tracking: Assign lecturers to modules

Entity Management
- Students: Register, update, view, and delete student records

- Lecturers: Manage lecturer profiles and departments

- Modules: Create academic modules with credit values

- Registrations: Enroll students in modules with duplicate prevention

- Module Assignments: Assign lecturers to modules

Advanced Features
🔍 Search functionality for students

📊 Dashboard with statistics

📱 Mobile-responsive interface

🎨 Modern UI with icons and cards

📈 Recent activity tracking
